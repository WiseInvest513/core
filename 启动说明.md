# 启动说明

## 快速启动

在项目目录下运行：

```bash
npm start
```

或者：

```bash
node server.js
```

## 访问地址

服务器启动后，在浏览器中访问：

**http://localhost:3001**

## 当前配置

- **端口**: 3001
- **VOO 价格**: $631.18（静态价格）
- **QQQ 价格**: $619.25（静态价格）
- **USD/CNY 汇率**: 7.15（静态汇率）

## 文件说明

- `index.html` - 主页面
- `style.css` - 样式文件
- `script.js` - 核心逻辑（包含价格和汇率配置）
- `server.js` - HTTP 服务器
- `package.json` - 项目配置
- `manifest.json` - PWA 配置

## 注意事项

- 使用 Node.js 原生 http 模块，无需安装额外依赖
- 所有价格和汇率都是写死在代码中的，不调用任何 API
- 确保端口 3001 未被占用

## 更新价格

如需更新价格，编辑 `script.js` 文件开头的常量：

```javascript
const CURRENT_STOCK_PRICES = {
    VOO: 631.18,  // 修改这里
    QQQ: 619.25   // 修改这里
};
const CURRENT_EXCHANGE_RATE = 7.15; // 修改这里
```
