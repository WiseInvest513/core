# Git 推送配置说明

## 快速推送方式

### 方式1：使用推送脚本（最简单）
```bash
./push.sh
```

### 方式2：直接命令
```bash
git push origin main
```

## 当前 Git 配置

你可以运行以下命令查看当前配置：
```bash
git config --list
```

## 如果推送失败，需要配置

### 配置代理（如果使用代理工具）
```bash
# 查看你的代理端口（通常是 7890 或 1080）
git config --global http.proxy http://127.0.0.1:7890
git config --global https.proxy http://127.0.0.1:7890
```

### 配置 SSH（推荐，更稳定）
```bash
# 1. 切换远程地址为 SSH
git remote set-url origin git@github.com:WiseInvest513/core.git

# 2. 确保已添加 SSH key 到 GitHub
# 查看公钥：cat ~/.ssh/id_ed25519.pub 或 cat ~/.ssh/id_rsa.pub
# 添加到 GitHub：https://github.com/settings/keys
```

### 配置凭据存储（避免每次输入密码）
```bash
# macOS
git config --global credential.helper osxkeychain

# 或者使用 cache（15分钟有效期）
git config --global credential.helper 'cache --timeout=3600'
```
